\# Mini Resume Collector API



\## Overview

This project is a FastAPI-based REST API that allows uploading candidate resumes and storing candidate metadata.  

The application supports filtering, searching, retrieving, and deleting candidate information.



The updated implementation includes \*\*database persistence using SQLite and SQLAlchemy\*\*, ensuring that data remains stored even after server restarts.



---



\## Technologies Used

\- Python 3.13

\- FastAPI

\- SQLAlchemy

\- SQLite

\- Uvicorn

\- Pydantic
